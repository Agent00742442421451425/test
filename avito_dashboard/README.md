# Avito Dashboard

Веб-приложение для учёта объявлений Авито, статистики и расчёта прибыли.

## Возможности

- **Объявления** — список ваших объявлений с Авито
- **Статистика** — просмотры, контакты за выбранный период
- **Прибыль** — оценка по оптово-розничному прайсу
- **Чек-листы** — рекомендации что делать дальше

## Установка

1. Установите зависимости:

```bash
pip install fastapi uvicorn openpyxl
```

2. Добавьте в `.env` ключи Авито API:

```
AVITO_CLIENT_ID=ваш_client_id
AVITO_CLIENT_SECRET=ваш_client_secret
```

Ключи получаете в разделе [«Для профессионалов» → «API»](https://www.avito.ru/professionals/api) на Авито.

3. (Опционально) Путь к прайс-листу при старте (полный путь к Excel):

```
AVITO_PRICE_FILE=/Users/artem/Downloads/Работа/Отчёты/Оптово-розничный прайс.xlsx
```

Или загрузите прайс через интерфейс на главной странице.

## Запуск

```bash
# Из корня проекта
uvicorn avito_dashboard.main:app --reload --port 8000
```

Откройте http://localhost:8000

## Прайс-лист

Формат Excel/CSV: колонки **артикул** (или код), **название**, **оптовая цена**, **розничная цена**.  
Можно загрузить через интерфейс или указать `AVITO_PRICE_FILE` в .env.
